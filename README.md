
# Simple Project Search Movie using Ajax OMDB API




## API Reference

#### OMDB API

```http
  http://www.omdbapi.com/?apikey=[yourkey]&
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `api_key` | `string` | **Required**. Your API key |
